## [Package Management] | [pip](https://pypi.org/project/pip/) | [github](https://github.com/pypa/pip)

```bash
$ pip list
$ pip show [package]
```

# Installation
```bash
$ pip install [package]
$ pip install [package] --no-index --find-links [path]
$ pip freeze > requirements.txt
$ pip install -r requirements.txt
```
