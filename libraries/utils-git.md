## Git

### Config

`Config Syntax`
```bash
$ git config [--global or --local] [category].[parameter] [value]
```

`list`
```bash
$ git config --list
```


#### Global Config
`~/.gitconfig`  

`registry`
```bash
$ git config --global user.name "user"
$ git config --global user.email "user@domain"
```



#### Local Config
`./.git/config`

```bash
$ git remote -v
```
